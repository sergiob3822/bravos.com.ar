Política de Privacidad

Última actualización: 18 de abril de 2026 · Versión 1.1

00

Responsable del Tratamiento

  • Responsable: Sergio Bravo
  • Sitio Web: bravos.com.ar
  • Contacto: admin@bravos.com.ar
  • Ubicación: Entre Ríos, Argentina
01

1. Información que Recolectamos

Para el funcionamiento del portafolio y las integraciones de API, podemos tratar los siguientes datos:

  • Datos Identificativos: Nombre, apellido y país de residencia.
  • Datos de Contacto: Correo electrónico, número de teléfono (+54 11 25421727) y perfiles de redes sociales o mensajería (Telegram, Instagram, Facebook).
  • Datos de Conectividad: Direcciones IP, metadatos de mensajes procesados a través de APIs de terceros (WhatsApp/Twilio) y registros de túneles de conexión para asegurar la estabilidad del servicio.
02

2. Finalidad del Tratamiento

Utilizamos la información recolectada para:

  • Gestionar y mostrar mi portafolio profesional y servicios de desarrollo.
  • Operar y administrar las integraciones de APIs para automatización de mensajería.
  • Establecer canales de comunicación directa con clientes y colaboradores a través de las redes sociales y bots proporcionados.
  • Garantizar la seguridad técnica del sitio, monitorizando posibles errores de conexión o accesos no autorizados.
03

3. Servicios de Terceros y APIs

Este portafolio utiliza servicios de terceros que pueden procesar datos de manera independiente:

  • Infraestructura y Túneles: Para garantizar la disponibilidad del sitio bajo el dominio bravos.com.ar.
  • APIs de Mensajería: Procesamiento de flujos de comunicación mediante herramientas de automatización.
  • Redes Sociales: El sitio contiene enlaces a perfiles externos (Instagram, Facebook, Telegram). Al acceder a estos enlaces, el tratamiento de sus datos se rige por las políticas de dichas plataformas.
04

4. Seguridad

Implementamos medidas técnicas para proteger sus datos personales. Se promueve el uso de métodos de autenticación seguros (como Passkeys) y protocolos de cifrado para todas las transferencias de datos entre las APIs y el servidor.

Dada la naturaleza de las conexiones de red, no podemos garantizar la invulnerabilidad absoluta, pero trabajamos constantemente en minimizar riesgos de "micro-cortes" o brechas de seguridad.

05

5. Derechos del Usuario

Usted tiene derecho a consultar, rectificar o solicitar la eliminación de sus datos personales. Para ejercer estos derechos, puede enviar una comunicación a admin@bravos.com.ar.

06

6. Contacto Directo

Para consultas adicionales sobre el tratamiento de datos en mis desarrollos o portafolio, puede contactarme a través de:
